## 🖼️ What is ArTodo?

> *"Every artist was first an amateur."* — Ralph Waldo Emerson

**ArTodo** is a premium to-do list app where productivity meets art. Each theme transforms your entire workspace — background animations, color palettes, typography, and UI accents — into a living canvas inspired by iconic art movements.

Built with **zero dependencies**. Pure HTML5 + CSS3 + Vanilla JavaScript.

## ✨ Features

<table>
<tr>
<td>

### 📋 Task Management
- ✅ Create, edit, delete tasks
- 🎯 Priority levels (High / Med / Low)
- 📁 Categories (Work, Personal, Health, Creative)
- 📅 Due dates with overdue alerts
- 📝 Nested subtasks with progress bars
- 🔄 Drag & drop reordering

</td>
<td>

### 🎨 Art Experience
- 🖼️ 6 animated canvas backgrounds
- ✨ Confetti burst on task completion
- 🔊 Subtle completion sound effects
- 🌊 Theme-specific micro-animations
- 🖱️ Mouse parallax on Starry Night
- 💫 Occasional shooting stars

</td>
</tr>
<tr>
<td>

### ⏱️ Productivity
- 🍅 Pomodoro Timer (25/5 min cycles)
- 📊 Statistics dashboard & weekly chart
- 🔥 Daily streak counter
- 📈 Completion rate tracking
- ⭕ Progress ring in header

</td>
<td>

### 🛠️ Technical
- 💾 LocalStorage persistence
- 🔍 Real-time search & filtering
- ⌨️ Keyboard shortcuts (`/` search, `Esc` close)

- 🚀 Zero dependencies, instant load

## 📁 Project Structure

```
artodo/
├── index.html          # Semantic HTML5 structure + modals
├── css/
│   ├── style.css       # Base styles, glassmorphism, animations
│   └── themes.css      # 6 art-inspired theme definitions
├── js/
│   └── app.js          # Store, ThemeManager, Pomodoro, Statistics, App
└── README.md
```

## 🏃 Quick Start

```bash
# Clone
git clone https://github.com/novariyaz/Artodo.git

# Open directly (no server needed!)
open index.html

# Or serve locally
npx -y http-server . -p 8080
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|

| `Esc` | Close any modal |
