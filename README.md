# 🎨 ArTodo — Artistic Task Manager

A premium, art-themed To-Do List app with 6 art movement-inspired themes.

## Features
- ✅ Task management with priorities, categories, due dates & subtasks
- 🎨 6 Art Themes: Starry Night, Ukiyo-e, Art Deco, Bauhaus, Impressionist, Cyberpunk
- 🖼️ Animated canvas backgrounds unique to each theme
- ⏱️ Pomodoro Timer (25/5 min focus/break)
- 📊 Statistics dashboard with weekly chart
- 🔍 Search, filter & drag-and-drop reorder
- 💾 LocalStorage persistence
- ✨ Confetti, sound effects & micro-animations

## Tech Stack
Pure HTML5 + CSS3 + Vanilla JavaScript — no frameworks, no build tools.
